Aggregation Framework¶

  • Mtodo extremamente flexvel de remodelar dados
  • Rica biblioteca de funes
  • Excelente para exploraes ad-hoc
  • Alternativa mais simples e fcil de programar ao MapReduce

$match¶

Filtra os documentos

  • Reduz a quantidade de documentos processados
  • Se utiliza de ndices

In [13]:
matching = {'$match': {'origem.estado': 'Gois', 
                       'peso': {'$gte': 1.3, '$lt': 10}}}
pipeline = [matching]
db.soja.aggregate(pipeline).next()

$project¶

Manipula os dados

  • Seleciona, cria ou renomeia campos
  • Injeta campos calculados
  • Pode criar sub-documentos

In [25]:
projecting = {
    '$project': {
        '_id': 0,
        'fazenda': '$origem.fazenda', 
        'peso': 1, 'safra': 1,
        'dim_x': '$dimensoes.x', 'dim_y': '$dimensoes.y',
        'grau_de_esfericidade': {'$abs': {'$subtract': [1, {'$divide': ['$dimensoes.x', '$dimensoes.y']}]}} 
    }
}
pipeline = [{'$sample': {'size': 10000}}, matching, projecting]
df = pd.DataFrame(list(db.soja.aggregate(pipeline)))
df.sample(2)

$group¶

Agrupa os documentos por algum campo

  • Agregaes como somas, mdias, counts, mximos e mnimos ($avg $sum $max $min)
  • Criao de arrays ou sets ($push ou $addToSet)

In [24]:
grouping = {
    '$group': {
        '_id': {'fazenda': '$fazenda', 'safra': '$safra'},
        'peso_medio': {'$avg': '$peso'},
        'peso_total': {'$sum': '$peso'}
    }
}
sorting = {'$sort': {'peso_medio': -1}}

pipeline = [matching, projecting, grouping, sorting]
df = pd.DataFrame(list(db.soja.aggregate(pipeline))).set_index("_id")
fig, ax = plt.subplots(1, 2, sharey=True)
df[['peso_medio']].plot(kind='barh', legend=False, title='Peso Mdio Gro de Soja', ax=ax[0]);
df[['peso_total']].plot(kind='barh', legend=False, title='Peso Total Gro de Soja', ax=ax[1]);

$unwind¶

Cria novos documentos a partir de campos de array

  • Utilizado para "destrinchar" campos listados
  • Em geral utilizado antes de um estgio $group

In [26]:
unwinding = {'$unwind': '$toxicos'}
db.soja.aggregate([unwinding]).next()

Outros Stages:¶

  • $sort, $skip $limit: paginao
  • $geoNear: query por proximidade por uso de coordenadas geogrficas
  • $let $map $reduce: criao de manipulaes customizadas
  • $facet: agregaes em paralelo
  • $lookup: JOIN !?

Retornando à Ciência de Dados - Machine Learning¶


  • Dados advindos do MongoDB podem ser utilizados na criao de modelos estatsticos sofisticados
  • MongoDB no uma engine de ML - mas pode facilitar muito o processo
  • Com o uso do Aggregation Framework, pode-se manipular e preparar os dados para algoritmos especializados
  • Via o driver pymongo pode-se facilmente integrar a base de dados com poderosas bibliotecas do stack cientfico do Python
